# 메인 화면 문구 수정

## 목표

메인 화면에 노출되는 `Current User Test` 문구를 `Current User`로 수정한다.
이번 플랜의 범위는 실제 사용자 진입 경로에서 표시되는 대상 문구 식별, 텍스트 치환, 그리고 화면 기준 회귀 확인으로 한정한다.

## 담당자

- 플래너
- 게임 디자이너
- 플랫폼 엔지니어
- 테스트 엔지니어

## 할 일

### 플래너

- [x] 이슈 요구사항을 `메인 화면의 Current User Test -> Current User 치환` 범위로 확정해 문서화한다.
- [x] 구현 역할과 검증 역할을 각각 `플랫폼 엔지니어`, `테스트 엔지니어`로 지정한다.

### 플랫폼 엔지니어

- [ ] 메인 화면에서 `Current User Test`가 렌더링되는 소스 위치를 확인하고 실제 사용자 진입 경로 기준 대상 문자열만 `Current User`로 수정한다.
- [ ] 문구 변경으로 인한 주변 UI 레이아웃 또는 다른 화면 문자열 회귀가 없도록 관련 상수/컴포넌트 영향을 함께 점검한다.

### 테스트 엔지니어

- [ ] 메인 화면에서 `Current User`가 표시되고 기존 `Current User Test` 문구가 남아 있지 않은지 확인한다.
- [ ] 변경 대상과 무관한 메인 화면의 기본 진입 동작에 이상이 없는지 수동 검증 결과를 기록한다.

# 첫 페이지에 아이디 / 비밀번호 로그인 기능 추가

## 목표

첫 페이지에서 익명 유저 생성만 제공하는 현재 진입점을, 어드민 접근을 위한 임시 아이디 / 비밀번호 로그인 흐름까지 포함하도록 확장한다.
Supabase 인증과 기존 `/admin` 권한 체크를 그대로 활용하면서, 로그인 성공 시 관리자 사용자가 첫 페이지에서 인증 상태를 확인하고 `/admin`으로 진입할 수 있어야 한다.

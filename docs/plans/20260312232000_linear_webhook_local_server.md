# Linear 웹훅 로컬 서버 추가

## 목표
Linear 웹훅 이벤트를 로컬 개발 환경에서 수신할 수 있도록 Node.js 기반의 경량 서버를 추가한다.
서버는 `3000` 포트에서 실행되며, 요청 본문을 수신하고 기본 응답 및 로그를 제공해야 한다.

또한 개발자가 별도 명령 조합 없이 서버를 바로 실행할 수 있도록 `package.json`에 전용 실행 명령을 추가한다.
초기 범위는 웹훅 수신 기반 확보에 한정하며, 서명 검증이나 Codex 자동 실행 연동은 후속 플랜으로 분리한다.

# 트럼프

## 목표

게임의 모든 재해와 축복의 원천. 플레이어가 직접 제어할 수 없는 활화산.

## 화면 위치

화면 상단 중앙, 황금 의자에 앉아 있음. 올림포스 제우스 포지션.

## 기분 게이지

보이지 않는 내부 수치. 직접 조작 불가, 간접 행동으로만 영향.

| 표정 | 상태 | 효과 |
|------|------|------|
| 😊 | 기분 좋음 | 관세 완화, 달러 비, 보너스 드랍 |
| 😤 | 짜증남 | 관세 폭탄, 빨간 번개 |
| 🤔 | 심심함 | **제일 위험.** 예측 불가 법안 발동 |
| 😴 | 잠듦 | **황금 타임.** 기상 시점 불확실 |

## 힌트 시스템 (관찰 플레이)

트럼프의 행동을 관찰하면 다음 법안 카테고리를 예측할 수 있다.

| 행동 | 예측 카테고리 |
|------|---------------|
| 스테이크 식사 | 식품/소비재 규제 |
| 골프 | 스포츠/레저 규제 |
| 거울 보기 | 외모/그림자/색상 규제 |
| TV 시청 | 미디어/통신 규제 |

관찰 자체가 게임 스킬. 숙련 플레이어는 행동 읽고 선제 비축.

# 트럼프

## 목표

게임의 중앙 엔티티. 시작존이며 제거되지 않는다. 트럼프는 악역이 아니라 그냥 트럼프의 일을 하는 대통령. 엉뚱한 정책이 실행되면서 주민들이 허둥지둥 대응하는 모습이 코미디를 만든다.

## 위치

게임의 중앙. 트럼프를 기준으로 마을과 주민이 주변에 배치된다.

## 기분 게이지

보이지 않는 내부 수치. 직접 조작 불가, 간접 행동으로만 영향.

| 표정 | 상태 | 효과 |
|------|------|------|
| 😊 | 기분 좋음 | 관세 완화, 달러 비, 보너스 드랍 |
| 😤 | 짜증남 | 관세 폭탄, 빨간 번개 |
| 🤔 | 심심함 | **제일 위험.** 예측 불가 법안 발동 |
| 😴 | 잠듦 | **황금 타임.** 기상 시점 불확실 |

## 힌트 시스템 (관찰 플레이)

트럼프의 행동을 관찰하면 다음 법안 카테고리를 예측할 수 있다.

| 행동 | 예측 카테고리 |
|------|---------------|
| 스테이크 식사 | 식품/소비재 규제 |
| 골프 | 스포츠/레저 규제 |
| 거울 보기 | 외모/그림자/색상 규제 |
| TV 시청 | 미디어/통신 규제 |

관찰 자체가 게임 스킬. 숙련 플레이어는 행동 읽고 선제 비축.

## 트럼프 탭

트럼프를 탭하면 소액 보상 + 반응 애니메이션. 어항 유리를 톡톡 치는 느낌.

- 탭할 때마다 소액 자원 드랍 + 트럼프 반응 (손 흔들기, 윙크, 엄지척 등)
- **너무 많이 탭하면 트럼프가 관심을 가짐** → 이벤트 트리거 확률 증가
- 기분별 반응이 다름:
  - 😊 기분 좋음 → 보너스 보상
  - 😤 짜증남 → 역효과 (벌금, 감봉 등)
  - 😴 수면 중 → **깨울 수 있음.** 고보상이지만 깨어난 트럼프는 예측 불가

> 대통령을 쿡쿡 찔러서 용돈 받는 9급 공무원. 건들면 안 되는데 건들고 싶은 긴장감.

# 플레이어 — 좌천당한 9급 공무원

## 목표

본청 복귀를 믿고 왔지만 상황은 계속 악화. 하다 보면 주민들한테 정 들어서 못 떠남.

## 설정

- 직급: 9급 주무관 (명찰 표시)
- 월급: 트럼프 예산 삭감 시 깎임. 가끔 현물 지급 (생선 3마리)
- 상사: 트럼프 (이름을 매번 틀리게 부름)
- 관사: 마을에서 제일 허름. 업그레이드 가능하지만 법안에 의해 원상복구 위험
- 핵심 업무: 할 일 = 공문서·민원 처리

## 감정적 보상

할 일 잘 처리하면 주민들이 변함:
쌀쌀맞은 태도 → 김치 갖다줌 → 이름 불러줌 → 생일 챙겨줌

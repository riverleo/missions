# 캐릭터 시스템

## 역할

캐릭터 수집과 조합 실험이 게임의 핵심 재미요소. 개별 특성과 조합 효과로 월드에 영향을 준다.

## 개별 특성

캐릭터마다 고유한 특성을 가지며, 월드 진입 시 또는 배치 중 효과를 발휘한다. 캐릭터 파일에서는 특성명과 값만 정의한다.

| 특성명 | 정의 |
|--------|------|
| 월드 진입 달러 보너스 | {캐릭터}가 월드에 진입 시 {금액}달러가 생긴다 |
| 달러 수입 배율 | {캐릭터}가 배치된 동안 달러 수입 속도가 {배율}배가 된다 |
| 행정명령 지연 | {캐릭터}가 배치된 동안 행정명령 완성 속도가 {비율}% 느려진다 |
| 행정명령 피해 감소 | {캐릭터}가 배치된 동안 행정명령의 부정적 효과가 {비율}% 줄어든다 |
| 시세 변동 완화 | {캐릭터}가 배치된 동안 금/비트코인 가격 변동 폭이 {비율}% 줄어든다 |
| 시세 정보 | {캐릭터}가 배치된 동안 다음 가격 변동 방향을 {초}초 전에 알려준다 |
| 트럼프 기분 보정 | {캐릭터}가 배치된 동안 트럼프 기분 하락 속도가 {비율}% 느려진다 |
| 캐릭터 유인 | {캐릭터}가 배치된 동안 새 캐릭터 등장 확률이 {비율}% 올라간다 |
| 조합 부스트 | {캐릭터}가 배치된 동안 조합 효과가 {비율}% 강화된다 |
| 위기 방어 | 행정명령 발동 시 {비율}% 확률로 효과를 완전히 무시한다 |

## 조합 효과

| 효과 | 설명 |
|------|------|
| 달러 수입 변동 | 캐릭터 간 관계(호감/적대)에 따라 달러 수입 속도가 달라짐 |
| 트럼프 기분 영향 | 배치된 캐릭터들의 트럼프 호감도 합산이 트럼프 기분에 영향 |

## Notes

- 효과 후보: 행정명령 힌트, 행정명령 대응력, 특수 이벤트(전용 대사/연출), 캐릭터 해금, 시세 힌트(금/비트코인), 건물 효과
- "캐릭터 해금"이 수집 루프와 가장 강하게 연결됨 — 조합 실험 자체가 수집의 수단
- 유튜브 컨텐츠 관점: 조합 효과가 눈에 보이는 변화로 나타나야 영상감이 됨

# 핵심 게임 루프

## 목표

평화기 → 폭발기 → 수습기 사이클을 반복하며 마을을 유지한다.

## 전개

### 1. 평화기

- 트럼프가 골프치거나, 스테이크 먹거나, 자고 있음
- 플레이어: 마을 인프라 건설, 자원 수집, 주민 케어
- **비축 타임:** 뭘 대비할지 모르니까 도박 → 이 선택이 전략의 핵심
- 비축 예시: 산소통, 사각형 바퀴, 모음 없는 사전, 그림자 차단막, 양초

### 2. 폭발기 (법안 발표)

- 트럼프가 갑자기 벌떡 일어남
- 컷신 연출 후 엉뚱한 법안/관세 발표
- 주민들에게 구체적인 피해 발생
- **같은 법안에 피해자와 수혜자가 동시에 존재** (핵심 딜레마)

### 3. 수습기

- 제한된 자원으로 누구를 먼저 도울지 선택
- 미리 비축한 물자가 있으면 피해 경감
- 없으면 주민 고통 → 빈사 → 사망

## 사망 시스템

| 단계 | 효과 |
|------|------|
| 고통 | 생산력 감소, 표정 변화 |
| 빈사 | 일 못함, 다른 주민 간호 필요 → 연쇄 생산력 저하 |
| 사망 | 퇴장. GDP 20% 손실 |

- 새 주민 충원 → 이민 허가 필요 → 트럼프 기분에 따라 열림/닫힘

## 자원 체계

- **GDP** — 한 명 감기 = 20% 폭락
- **국제적 인지도** — 너무 낮으면 무역 불가, 너무 높으면 강대국 간섭
- **국민 사기(Morale)** — 한 명 삐지면 국가 위기

## 노트

- 한 사이클에서 비축 성공/실패 체감이 명확해야 한다.
- 법안은 즉사형보다 연쇄적 선택 압박형이 코미디와 전략성에 유리하다.

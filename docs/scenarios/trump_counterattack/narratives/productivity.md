# 할 일 ↔ 게임 매핑

## 목표

생산성 앱(할 일 관리)과 게임 행동을 1:1로 연결한다.

## 매핑 테이블

| 현실 | 게임 내 표현 |
|------|-------------|
| 할 일 등록 | 민원 접수 — 주민이 창구에 와서 하소연 |
| 할 일 완료 | 빨간 도장 **꽝!** + 서류 날아감 + 주민 "감사합니다!" |
| 할 일 미루기 | 서류 위에 먼지 쌓임 + 주민 표정 점점 어두워짐 |
| 할 일 삭제 | "이 민원은 각하되었습니다" — 주민이 슬프게 돌아감 |
| 중요 할 일 | 빨간 딱지 붙은 긴급 공문 (트럼프 직인) |
| 기한 지남 | 감사원 경고 도장 찍힘 |
| 앱 안 열기 | 책상이 서류로 파묻힘 + 주민들 접이식 의자 펴고 대기 |
| 앱 열기 | "출근" 효과 — 관공서 문 삐걱 소리 + 형광등 깜빡 |

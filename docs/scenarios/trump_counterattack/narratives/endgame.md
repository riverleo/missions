# 엔드게임 콘텐츠

## 목표

1급 승진 이후에도 플레이어가 게임(= 할 일 관리)을 지속할 동기를 제공한다.
급수 시스템 종점(1급)에서 끝나지 않고, 축적된 선택의 결과가 드러나는 구간을 설계한다.

---

## 전개: 1급 이후의 세계

### 1급 도달 조건

- 급수 시스템을 통해 9급 → 1급까지 승진 완료
- 1급 도달 시점부터 엔드게임 페이즈 진입

### 1급의 축복과 저주

| 항목 | 내용 |
|------|------|
| 권한 | 법안 사전 열람, 의견서 제출, 예산 자율 편성, 자치권 |
| 대가 | 트럼프 직통 전화(새벽 3시), 특별 임무 빈도 최대, 언제든 해고 가능 |
| 핵심 긴장 | "내 최측근"이라는 호칭 = 가장 높은 곳에서 떨어질 준비가 된 자리 |

---

## 엔드게임 페이즈: "정상의 폭풍"

1급 도달 이후 새로운 이벤트 계층이 열린다.

### 페이즈 1 — 트럼프의 시험

- 트럼프가 플레이어를 "진짜 내 사람인지" 시험하는 특수 퀘스트 연쇄 발동
- 선택지가 극단적으로 양분됨:
  - A: 트럼프 충성 → 마을에 직접적 피해를 주는 명령 수행
  - B: 마을 편 → 트럼프 명령 불이행, 강등/해고 위험
  - C: 양쪽 속이기 → 성공 시 최선, 실패 시 최악(양쪽 모두 적)
- 이 시험 결과가 엔딩 분기의 주요 변수

### 페이즈 2 — 최종 법안: "위대한 마무리 법"

- 트럼프가 "역대 최고의 법안"을 준비 중이라는 힌트가 수차례 등장
- 이 법안은 마을 존망을 좌우하는 규모
- 법안 내용은 그동안 플레이어가 가장 많이 비축한 자원을 정확히 타격하도록 설계됨
  → "아무리 준비해도 빗나간다"는 블랙코미디의 정점
- 최종 법안 대응 결과 + 누적 선택에 따라 엔딩 분기

---

## 엔딩 분기

5개의 엔딩이 존재하며, 각각 두 축의 조합으로 결정된다:
- **트럼프 호감도** (높음 / 낮음)
- **마을 상태** (번영 / 생존 / 폐허)

### 엔딩 A — "영광의 복귀" (트럼프 호감 高 + 마을 번영)

> 조건: 트럼프 호감도 최상위 + 주민 전원 생존 + GDP 기준치 이상

- 본청 복귀 발령이 내려온다
- 주민들이 환송 잔치를 열지만 얼굴이 복잡하다
- 마을을 떠나는 날, 트럼프가 전화: "다음 마을도 있어. 더 작은 데."
- **묘비명 스타일 엔딩 자막:** *"승진은 했다. 그런데 왜 기쁘지 않지?"*
- 블랙코미디 포인트: 복귀해봤자 더 큰 지옥이 기다리고 있음

### 엔딩 B — "조용한 독립" (트럼프 호감 低 + 마을 번영)

> 조건: 트럼프 호감도 최하위 + 주민 전원 생존 + 국제 인지도 기준치 이상

- 마을이 사실상 자치를 이루고, 트럼프가 이 마을의 존재를 잊어버린다
- 플레이어는 9급 명찰을 그대로 달고 있지만 실질적 자치 수장
- 트럼프 화면 상단에서 이 마을 방향을 안 봄 → 평화
- **엔딩 자막:** *"관세는 잊혀지고, 마을은 남았다."*
- 블랙코미디 포인트: 최선의 결과가 "무시당하는 것"

### 엔딩 C — "트럼프의 오른팔" (트럼프 호감 高 + 마을 폐허)

> 조건: 트럼프 호감도 최상위 + 주민 2명 이상 사망 + 법안 시행률 최상

- 트럼프가 플레이어를 "본부"로 발탁
- 마을은 폐허, 남은 주민이 플레이어를 원망하며 배웅
- 본부에서의 첫 업무: 다른 소국에 관세 폭탄 법안 전달
- **엔딩 자막:** *"결국 나도 위에서 내려보내는 쪽이 되었다."*
- 블랙코미디 포인트: 가해자가 되는 순환 구조

### 엔딩 D — "영원한 9급" (트럼프 호감 低 + 마을 생존)

> 조건: 트럼프 호감도 하위 + 강등 3회 이상 + 주민 과반 생존

- 1급까지 올라갔다가 다시 9급으로 강등
- 접이식 책상, 잉크 번지는 도장이 돌아옴
- 주민들이 김치, 감자, 양말을 가져다줌
- 트럼프가 다시 이름을 모름 → 오히려 안전
- **엔딩 자막:** *"급수는 돌아왔지만, 이상하게 책상이 따뜻하다."*
- 블랙코미디 포인트: 가장 낮은 자리가 가장 인간적

### 엔딩 E — "폐허 속 마지막 공무원" (호감도 무관 + 마을 폐허)

> 조건: 주민 4명 이상 사망 + GDP 최저

- 마을에 아무도 남지 않았다
- 플레이어 혼자 빈 관공서에 앉아 서류를 정리하고 있다
- 도장을 찍을 민원이 없다
- 트럼프 화면에서 이 마을이 지도에서 사라짐
- **엔딩 자막:** *"서류함은 비어 있다. 할 일이 없다는 건 이런 기분이었다."*
- 블랙코미디 포인트: 할 일 관리 앱인데 할 일이 사라진 아이러니

## 엔딩 후 콘텐츠

### 뉴게임+ : "다른 마을로 발령"

- 클리어 후 새로운 마을에서 다시 9급으로 시작
- 이전 회차 보너스:
  - 도장 스킨 해금 (이전 엔딩별 고유 도장)
  - 시작 비축 아원 1종 선택 가능
  - 이전 마을 주민이 가끔 편지를 보내옴 (할 일 완료 시 랜덤 등장)
- 새 마을 차별점:
  - 주민 구성이 다름 (새 캐릭터 풀)
  - 지역 특산물/수출품 변경 → 법안 영향이 달라짐
  - 트럼프 기분 패턴 변형

### 자유 모드: "만년 공무원"

- 엔딩 직전 세이브에서 분기, 엔딩을 보지 않고 계속 플레이
- 최종 법안이 발동하지 않는 상태로 무한 사이클
- 생산성 앱으로서의 본래 용도에 충실한 모드
- 모든 법안 풀이 열려서 다양한 조합 경험 가능

### 도전 과제 시스템

| 도전 과제 | 조건 | 보상 |
|-----------|------|------|
| 무결점 공무원 | 엔딩까지 강등 0회 | 금색 명찰 스킨 |
| 인간 방패 | 주민 전원 생존으로 엔딩 | 주민 감사 편지 컬렉션 |
| 트럼프 위스퍼러 | 힌트 시스템 정확도 90% 이상 | 트럼프 행동 예측 UI 강화 |
| 사각형 세계 | 둥근 물체 금지 법안 5회 생존 | 사각형 바퀴 커스텀 마을 장식 |
| 무호흡 달인 | 공기 관세 법안 3회 연속 무사 통과 | 산소통 영구 할인 |
| 오타 해석가 | 새벽 트윗 법안 전부 올바르게 해석 | 태수 전용 패시브 강화 |
| 5엔딩 컴플리트 | 모든 엔딩 확인 | 히든 엔딩 해금 |

### 히든 엔딩 — "공무원의 꿈"

> 조건: 5엔딩 모두 클리어 후 뉴게임+ 진입 시 특수 선택지 등장

- 2회차 시작 시 "사표를 쓸까요?" 선택지
- 사표 선택 → 공무원을 그만두고 마을 주민이 됨
- 플레이어가 직접 민원을 넣는 입장이 됨
- 새 NPC 공무원이 부임해서 플레이어의 민원을 처리
- **엔딩 자막:** *"창구 반대편에서 보니, 이 의자가 더 편하다."*
- 블랙코미디 포인트: 관점 역전, 결국 어디에 있든 시스템 안

## 분기/조건 요약

```
                    ┌─ 마을 번영 → 엔딩 A "영광의 복귀"
    트럼프 호감 高 ─┤
                    └─ 마을 폐허 → 엔딩 C "트럼프의 오른팔"

                    ┌─ 마을 번영 → 엔딩 B "조용한 독립"
    트럼프 호감 低 ─┤
                    └─ 마을 생존 → 엔딩 D "영원한 9급"

    호감도 무관     ── 마을 폐허 → 엔딩 E "폐허 속 마지막 공무원"

    5엔딩 컴플리트  ── 뉴게임+ → 히든 엔딩 "공무원의 꿈"
```

## 노트

- 엔딩은 단순 보상이 아니라 "플레이어의 선택이 어떤 의미였는지"를 보여주는 구간이어야 한다.
- 모든 엔딩에 블랙코미디 톤의 묘비명/자막을 붙여서 여운을 남긴다.
- 생산성 앱 본연의 목적(할 일 관리)은 자유 모드에서 엔딩 이후에도 계속 작동해야 한다.
- 뉴게임+의 "다른 마을"은 향후 시나리오 확장(새 시나리오 슬러그)으로 연결될 수 있다.
- 도전 과제는 게임 내 실제 시스템 규칙과 교차 확인 필요 (게임 디자이너 검토 대상).

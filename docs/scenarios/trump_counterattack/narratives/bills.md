# 법안 시스템

## 목표

트럼프가 발표하는 엉뚱한 법안/관세가 게임의 핵심 이벤트.
이름은 거창, 내용은 재앙. 트럼프식 자화자찬 네이밍.

## 법안 네이밍 규칙

- 항상 자화자찬형 제목
- 예: "아름답고 위대한 젓가락 자유 보호법", "미국을 다시 둥글지 않게 만드는 법"

## 발표 연출

기본: 화면 암전 → 스포트라이트 → 금색 글씨 낙하 → 팡파레 + 독수리 울음 + 폭죽 + 성조기 → 주민 정지 표정

### 바리에이션 (트럼프 기분별)

| 타입 | 연출 | 법안 특징 |
|------|------|-----------|
| 분노 발표 | 빨간 화면 + 책상 내려침 + 불타는 글씨 | 허점 많음 → 회피 가능 |
| 득의양양 발표 | 금색 비 + 트럼프 얼굴 동전 | 촘촘, 회피 어려움 |
| 심심풀이 발표 | 가벼운 휘파람 | 가장 엉뚱한 법안 |
| 새벽 3시 트윗 | 어두운 화면 + 스마트폰 불빛 | 오타 섞임 → 해석 논쟁 |
| 골프장 발표 | 골프공 궤적 따라 법안 날아옴 | 방향 틀어질 수도 |

## 법안 목록

| 법안 | 효과 | 블랙코미디 포인트 |
|------|------|-------------------|
| 둥근 물체 수출입 금지 | 바퀴 사용 금지, 경제 마비 | "타원형은 둥근 건가요?" 법적 논쟁 |
| 그림자세 도입 | 그림자 면적 비례 과세 | 정오에만 활동, 키 큰 주민 최대 피해 |
| 외국산 공기에 관세 200% | 숨 쉬기 = 과세 | 산소통 미비축 시 캐릭터 사망 |
| 화요일 폐지 | 한 주 6일 | 화요일생 생일 소멸 |
| 감정 수출 규제 | 외국인에게 웃으면 벌금 | 관광업 궤멸 |
| 알파벳 'E' 사용료 | 이름에 E 포함 시 파산 위기 | 개명 러시 |
| 중력세 | 무거울수록 세금 | 다이어트 = 생존전략 |
| 색깔 사용 규제법 | 마을 흑백화 | 비축물자 무용지물 |

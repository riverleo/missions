# 트럼프의 역습

## 기본 정보

| 항목 | 값 |
|------|-----|
| 슬러그 | `trump_counterattack` |
| 톤 | 블랙코미디 |
| 배경 | 현대. 트럼프가 대통령인 가상의 초소형 국가. 국민 6명, 정부 조직은 대국 수준 |
| 주민 규모 | 최대 6명 |

## 핵심 콘셉트

- **트럼프 = 게임의 중앙.** 시작존이자 제거되지 않는 핵심 엔티티
- 트럼프는 악역이 아님. 그냥 트럼프의 일을 할 뿐. **엉뚱한 정책이 실행되면서 주민들이 허둥지둥 대응하는 모습**이 코미디의 원천
- 트럼프 주변으로 마을과 주민이 배치됨

## 플레이 방식

화면 한쪽에 켜놓고 필요할 때 할 일을 추가하면서 게임을 구경한다. 가끔 머리 식힐 때 트럼프를 탭하거나 아이템·건물을 배치하는 **어항형 아이들러**.

- 할 일 완료 → 자원 생산 → 마을 성장
- 마을이 성장하면 트럼프 눈에 띔 → 법안/이벤트 발생
- 법안의 결과가 마을에 펼쳐지는 것을 구경
- 안 해도 돌아가지만, 하면 더 재밌어지는 구조

## 시스템 구조

| 시스템 | 한줄 요약 |
|--------|-----------|
| 급수 (9급→1급) | 승진은 축복이자 저주. 권한 증가와 함께 트럼프 간섭도 증가 |
| 트럼프 호감도 | 법안 대응 선택에 따라 변동. 높으면 승진+특별임무, 낮으면 주민 협조+비공식 루트 |
| 마을 여론 | 친트럼프↔반트럼프 스펙트럼. 양 극단 모두 재앙, 중간 유지가 최적 |
| 법안·관세 | 엉뚱한 법안 발표 → 비축 아이템으로 대응. 트럼프 관찰이 힌트 |
| 트럼프 기분 | 보이지 않는 내부 수치. 표정으로만 추측. 심심함이 최고 위험 |
| 캐릭터 사망 | 고통→빈사→사망 단계. 1명 사망 = GDP 20% 손실 |
| 자원 | GDP, 국제 인지도, 국민 사기(Morale) |
| 할일↔게임 매핑 | 할 일 등록=민원 접수, 완료=도장 꽝!, 미루기=먼지 쌓임 |

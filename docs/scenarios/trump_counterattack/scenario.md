# 트럼프의 역습

## 기본 정보

- 슬러그: `trump_counterattack`
- 장르: 아이들(Idle) 게임
- 톤: 블랙코미디
- 배경: 현대, 관세 전쟁 피해를 받는 초소형 국가(마을 단위)
- 플레이 가능 주민 수: 최대 6명
- 플레이어 역할: 가상의 소국 행정 공무원

## 핵심 콘셉트

화면 상단 중앙에는 황금 의자에 앉은 거대한 트럼프가 있고, 화면 하단에는 작은 마을과 주민이 있다.
트럼프는 축복과 재해를 동시에 내려보내는 활화산 같은 존재이며, 상단의 화려함과 하단의 참상을 대비시키는 연출이 핵심 유머 포인트다.

## 화면 구성

1. 상단: 거대한 트럼프, 기분 게이지 표정 변화, 법안 발표 연단
2. 하단: 마을 시설, 주민 상태, 비축 창고
3. 중앙 이벤트 레이어: 법안 발표 컷신/재난 연출

## 핵심 게임 루프

1. 평화기

- 트럼프가 골프, 식사, 수면 등 일상 행동을 반복한다.
- 플레이어는 인프라를 건설하고 자원을 모으며 주민을 관리한다.
- 다음 재난을 예측해 비축 아이템을 선택한다.
- 비축 예시: 산소통, 사각형 바퀴, 모음 없는 사전, 그림자 차단막, 양초

2. 폭발기(법안 발표)

- 트럼프가 갑자기 일어나 법안/관세를 발표한다.
- 발표 즉시 주민/마을에 구체적인 페널티가 적용된다.

3. 수습기

- 제한된 자원으로 구조 우선순위를 결정한다.
- 비축 성공 시 피해를 경감한다.
- 실패 시 고통 -> 빈사 -> 사망으로 진행된다.

## 법안 발표 연출

기본 연출:

- 화면 암전 -> 트럼프 스포트라이트 -> 금색 타이포 낙하
- 팡파레 + 독수리 울음 + 폭죽 + 성조기 모션
- 카메라 하단 패닝 후 주민 정지 표정 노출

발표 바리에이션:

- 분노 발표: 붉은 화면, 책상 타격, 불타는 자막 (허점 많은 법안)
- 득의양양 발표: 금색 파티클, 코인 이펙트 (회피 어려운 법안)
- 심심풀이 발표: 휘파람 BGM (가장 엉뚱한 법안)
- 새벽 3시 트윗: 스마트폰 광원만 남은 어두운 화면 (오타 법안)
- 골프장 발표: 골프공 탄도형 법안 등장 (탄도 오차 가능)

## 법안 네이밍 규칙

- 항상 자화자찬형 제목
- 이름은 거창하고 효과는 재앙
- 예시: `아름답고 위대한 젓가락 자유 보호법`, `미국을 다시 둥글지 않게 만드는 법`

## 법안/관세 예시

| 법안                    | 게임 효과                     | 코미디 포인트               |
| ----------------------- | ----------------------------- | --------------------------- |
| 둥근 물체 수출입 금지   | 바퀴/공놀이 봉쇄, 물류 급감   | `타원형도 둥근가요?` 논쟁   |
| 그림자세 도입           | 그림자 면적 비례 과세         | 정오 출근 러시              |
| 외국산 공기에 관세 200% | 호흡 자원 소비 폭증           | 산소통이 실질 화폐화        |
| 화요일 폐지             | 주간 사이클 붕괴              | 생일/급여일 혼란            |
| 감정 수출 규제          | 웃음/친절 상호작용 제한       | 관광 산업 즉시 위축         |
| 알파벳 E 사용료         | 이름/문서 유지비 증가         | 강제 개명 유행              |
| 중력세                  | 체중 기반 비용 증가           | 다이어트가 국방 전략        |
| 색깔 사용 규제법        | 화면 흑백화, 일부 아이템 무효 | 화려한 마을이 단숨에 무채색 |

## 트럼프 기분 게이지

- 보이지 않는 내부 수치로 동작하며 직접 조작은 불가
- 플레이어는 간접 행동(비축/대응/이벤트 처리)으로만 영향
- 표정 상태:
  - 😊 기분 좋음: 완화 이벤트, 보너스 드랍
  - 😤 짜증: 고강도 관세/재난 발동
  - 🤔 심심: 예측 불가한 법안 발동(최고 위험)
  - 😴 수면: 안전 시간대, 단 기상 시 급반전 가능

## 힌트 시스템 (관찰 플레이)

| 트럼프 행동   | 예측 카테고리         |
| ------------- | --------------------- |
| 스테이크 식사 | 식품/소비재 규제      |
| 골프          | 스포츠/레저 규제      |
| 거울 보기     | 외모/그림자/색상 규제 |
| TV 시청       | 미디어/통신 규제      |

관찰 정확도가 높을수록 선제 비축 성공률이 증가한다.

## 사망 시스템

상태 단계:

- 고통: 생산력 저하, 표정 변화
- 빈사: 작업 불가에 가까움, 간호 인력 소모
- 사망: 캐릭터 이탈, 경제/사기 복합 손실

게임 영향:

- 주민 1명 사망 시 GDP 20% 손실
- 신규 주민 충원은 이민 허가 이벤트에 종속됨

## 자원 체계

- GDP: 인프라/생산력 핵심 지표
- 국제적 인지도: 무역/외교 이벤트 개방 조건
- 국민 사기(Morale): 생산 효율과 이탈 확률에 영향

## 밸런스 메모

- 한 사이클(평화기 -> 폭발기 -> 수습기)에서 비축 성공/실패 체감이 명확해야 한다.
- 법안은 즉사형보다 연쇄적 선택 압박형이 코미디와 전략성에 유리하다.
- 기분 게이지는 예측 가능성과 랜덤성을 함께 제공해야 한다.

# 트럼프의 역습

## 기본 정보

| 항목 | 값 |
|------|-----|
| 슬러그 | `trump_counterattack` |
| 톤 | 블랙코미디 |
| 배경 | 트럼프가 대통령인 초소형 국가. 정부는 대국 수준 |

## 핵심 콘셉트

- **캐릭터 수집과 조합이 게임의 주요 재미요소.** 포켓몬 카드처럼 캐릭터를 모으고 조합을 짜는 것이 핵심
- **할 일 완료 → 캐릭터들이 환호하고 기뻐하는 것**이 핵심 보상
- **트럼프 = 게임의 중앙** 시작존이자 제거되지 않는 핵심 엔티티
- 트럼프는 악역이 아님. 그냥 트럼프의 일을 할 뿐. **엉뚱한 정책이 실행되면서 주민들이 허둥지둥 대응하는 모습**이 코미디의 원천

## 플레이 방식

화면 한쪽에 켜놓고 필요할 때 할 일을 추가하면서 게임을 구경한다. 가끔 머리 식힐 때 트럼프를 탭하거나 아이템·건물을 배치하는 **방치형 게임**.

- 할 일 완료 → 자원 생산 → 마을 성장
- 급수가 올라갈수록 트럼프 간섭 증가 → 더 강한 행정명령 발생
- 행정명령 완성 시 게임 일시정지 → 플레이어가 통과/거절 선택 → 결과가 마을에 펼쳐지는 것을 구경
- 적극적으로 플레이할수록 더 재밌어지는 구조

## 시스템 구조

| 시스템 | 한줄 요약 |
|--------|-----------|
| 급수 (9급→1급) | 승진 시 건물 잠금해제·트럼프 탭 보상 증가와 함께 트럼프 간섭도 증가 |
| 트럼프 호감도 | 캐릭터마다 -10~+10 값. 배치된 캐릭터들의 호감도 합산이 트럼프 기분에 영향 |
| 캐릭터 조합 | 캐릭터 간 관계(호감/적대)에 따라 달러 수입 속도가 달라짐 |
| 행정명령 | 엉뚱한 행정명령 발표 → 비축 아이템으로 대응. 트럼프 관찰이 힌트 |
| 트럼프 기분 | 탭(관심)하면 좋아짐, 방치하면 나빠짐. 기분에 따라 행정명령 완성 속도 변동 |
| 자원 | 달러, 금, 비트코인. 달러로 금/비트코인을 매매할 수 있고, 트럼프 행동에 따라 가격이 폭등·폭락하는 도박 시스템 |
